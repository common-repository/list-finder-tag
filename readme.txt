Add List Finder tracking tag in HEAD tag.

== Description ==
 
You can automatically insert List Finder tracking tag with your List Finder ID in HEAD tag area.
 
== Installation ==
 
You can install this plugin directly from your WordPress dashboard:
 
 1. Go to the *Plugins* menu and click *Add New*.
 2. Search for *listfinder*.
 3. Click *Install Now* next to the *List Finder Tag* plugin.
 4. Activate the plugin.
 5. Configure your List Finder ID in *List Finder Tag* in *Setting* menu.
